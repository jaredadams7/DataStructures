Author: Jared Adams

This project implements an ADT, array based queue. When ran, the commands 
listed in QueType.in will be ran against the queue implementation. The 
output will be written to output.txt.

This project includes:
  -implements ADT, generic array based queue
  -circular, floating queue with Enqueue and Dequeue complexity O(1)
  -generic data type implementation
  
To compile:
  $ make

To run:
  $ ./Que

To clean:
  $ make clean
